# 两有限长序列之间的卷积

我们知道，两有限长序列之间的卷积可以用圆周卷积代替，假设两有限长序列的长度分别为MMM和NNN,那么卷积后的长度为L=M+N−1L=M+N-1L=M+N−1,那么用圆周卷积计算线性卷积的具体过程为:

  1. 首先将两序列在尾部补零，延拓成长度为 L=M+N−1L=M+N-1L=M+N−1 的序列
  2. 将两序列进行圆周卷积，卷积后的结果即为线性卷积的结果

# 有限长序列与无限长序列卷积

或者说有限长序列与另一长度远大于其长度的序列进行卷积，如果按照上面直接用DFTDFTDFT计算的话，有两个问题。

  1. 必须知道无限长序列的全部元素，才能进行计算
  2. 用DFTDFTDFT计算卷积可能还不如直接进行卷积运算来得快

为解决上述的问题，可以将无限长序列划分为短序列，将短序列与有限长序列进行卷积，然后对结果进行处理，主要由两种方法:重叠相加法和重叠保留法。

# 重叠保留法

同样将x[n]x[n]x[n]以长度NNN进行划分，一般取N>MN>MN>M,这时以NNN点进行圆周卷积。实际卷积的长度l=N+M−1l=N+M-1l=N+M−1,由圆周卷积与线性卷积的关系，知圆周卷积的后2N−l=N−M+12N-l=N-M+12N−l=N−M+1个点与线性卷积的结果是一致的。

取

xm[n]={x[n+mL],0≤n≤N−10,otherx_m[n]= \begin{cases} x[n+mL], &0 \leq n \leq N-1 \\ 0, &other \end{cases} x​m​​[n]={​x[n+mL],​0,​​​0≤n≤N−1​other​​

将xm[n]x_m[n]x​m​​[n]与h[n]h[n]h[n]进行NNN点圆周卷积得到ym[n]y_m[n]y​m​​[n]，只取后N+M−1N+M-1N+M−1个点，其余重叠的前M−1M-1M−1个点舍弃(保留)。然后进行移位相加，得到的结果就是进行线性卷积的结果。

由上图知，要使得到的结果表示y[n]y[n]y[n],应使得L−M−2=N−2+1⇒L=N−M+1L-M-2=N-2+1 \Rightarrow L=N-M+1L−M−2=N−2+1⇒L=N−M+1。

同理上面提到的圆周卷积均可用DFTDFTDFT进行实现。
