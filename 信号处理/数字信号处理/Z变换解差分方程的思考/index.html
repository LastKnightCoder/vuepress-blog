一般求解这种题目的思路很清晰，首先根据特征方程求出特征根，从而得出零输入解的形式，但是这个时候给的条件是y(0)y(0)y(0)和y(1)y(1)y(1)，而不是yzi(0)y_{zi}(0)y​zi​​(0)和yzi(1)y_{zi}(1)y​zi​​(1)，这意味着此时我们不能直接得出零输入解的系数。这个时候我们可以求出系统的零状态响应，然后可以得出yzs(0)y_{zs}(0)y​zs​​(0)和yzs(1)y_{zs}(1)y​zs​​(1)，然后根据yzi(0)=y(0)−yzs(0)y_{zi}(0) = y(0) - y_{zs}(0)y​zi​​(0)=y(0)−y​zs​​(0)，yzi(1)=y(1)−yzs(1)y_{zi}(1) = y(1) - y_{zs}(1)y​zi​​(1)=y(1)−y​zs​​(1)，然后得出零输入响应的系数，然后将零输入响应和零状态响应相加得到全响应。

上面的思路很清晰，但是却是有点麻烦，我们可以根据

上面的式子是我在答案(不一定正确的答案)上看到的，但是你没有发现有没有一丝丝的不对，怎么

因为x(−1)=x(−2)=0x(-1) = x(-2) = 0x(−1)=x(−2)=0，所以后面就没有"尾巴"，然后两边同时乘以z2z^2z​2​​得到

z2Y(z)−710z(Y(z)+y(−1)z)+110(Y(z)+y(−1)z+y(−2)z2)=7z2X(z)−2zX(z)z^2Y(z) - \dfrac{7}{10}z(Y(z) + y(-1)z) + \dfrac{1}{10}(Y(z) + y(-1)z + y(-2)z^2) = 7z^2X(z) - 2zX(z) z​2​​Y(z)−​10​​7​​z(Y(z)+y(−1)z)+​10​​1​​(Y(z)+y(−1)z+y(−2)z​2​​)=7z​2​​X(z)−2zX(z)

# 总结

其实根据题目给的初始条件不同，我们的解法也不相同，这里总结一下如果给出不同的初始条件应当采取什么解法(以二阶为例)：

  1. 给出y(0),y(1)y(0), y(1)y(0),y(1)，就是我上面提到的情况，有两种方法

    1. 分别求零输入响应和零状态响应，此时零输入响应的系数需要求出零状态响应后才能知道
    2. 直接进行ZZZ变换

  2. 给出yzi(0),yzi(1)y_{zi}(0), y_{zi}(1)y​zi​​(0),y​zi​​(1)，同样也有两种方法，不过与上面有所不同(实际上是简单了)

    1. 分别求零输入响应和零状态响应，此时由于直接给出了yzi(0),yzi(1)y_{zi}(0), y_{zi}(1)y​zi​​(0),y​zi​​(1)，所以可以直接求出零输入响应的系数
    2. 直接进行ZZZ变换，此时的形式与上面又有所不同，这时不是

    z2(Y(z)−y(0)−y(1)z−1)−710z(Y(z)−y(0))+110Y(z)=7z2(X(z)−x(0)−x(1)z−1)−2z(X(z)−x(0))z^2(Y(z) - y(0) - y(1)z^{-1}) - \dfrac{7}{10}z(Y(z) - y(0)) + \dfrac{1}{10}Y(z) = 7z^2(X(z) - x(0) - x(1)z^{-1}) - 2z(X(z) - x(0)) z​2​​(Y(z)−y(0)−y(1)z​−1​​)−​10​​7​​z(Y(z)−y(0))+​10​​1​​Y(z)=7z​2​​(X(z)−x(0)−x(1)z​−1​​)−2z(X(z)−x(0))

    而是

    z2(Y(z)−yzi(0)−yzi(1)z−1)−710z(Y(z)−yzi(0))+110Y(z)=7z2X(z)−2zX(z)z^2(Y(z) - y_{zi}(0) - y_{zi}(1)z^{-1}) - \dfrac{7}{10}z(Y(z) - y_{zi}(0)) + \dfrac{1}{10}Y(z) = 7z^2X(z) - 2zX(z) z​2​​(Y(z)−y​zi​​(0)−y​zi​​(1)z​−1​​)−​10​​7​​z(Y(z)−y​zi​​(0))+​10​​1​​Y(z)=7z​2​​X(z)−2zX(z)

  3. 给出y(−2),y(−1)y(-2), y(-1)y(−2),y(−1)，同样两种解法，因为yzs(−2)=yzs(−1)=0y_{zs}(-2) = y_{zs}(-1) = 0y​zs​​(−2)=y​zs​​(−1)=0，所以yzi(−2)=y(−2),yzi(−1)=y(−1)y_{zi}(-2) = y(-2), y_{zi}(-1) = y(-1)y​zi​​(−2)=y(−2),y​zi​​(−1)=y(−1)，这时同第二种情况是一样的了

    1. 分别求零输入响应和零状态响应，过程同第二种情况一样

    2. 直接进行ZZZ变换，此时的形式为

      Y(z)−710z−1(Y(z)+y(−1)z)+110z−2(Y(z)+y(−1)z+y(−2)z2)=7X(z)−2z−1X(z)Y(z) - \dfrac{7}{10}z^{-1}(Y(z) + y(-1)z) + \dfrac{1}{10}z^{-2}(Y(z) + y(-1)z + y(-2)z^2) = 7X(z) - 2z^{-1}X(z) Y(z)−​10​​7​​z​−1​​(Y(z)+y(−1)z)+​10​​1​​z​−2​​(Y(z)+y(−1)z+y(−2)z​2​​)=7X(z)−2z​−1​​X(z)
