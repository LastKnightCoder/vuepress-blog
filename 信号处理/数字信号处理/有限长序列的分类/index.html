# 基于共轭对称的分类

模运算给出了对称的一种定义

x[n]=xcs[n]+xca[n]x[n]=x_{cs}[n]+x_{ca}[n] x[n]=x​cs​​[n]+x​ca​​[n]

圆周共轭对称

xcs[n]=12(x[n]+x∗[<−n>N]),0≤n≤N−1x_{cs}[n]=\frac{1}{2}(x[n]+x^{*}[<-n>_N]), \, 0\leq n \leq N-1 x​cs​​[n]=​2​​1​​(x[n]+x​∗​​[<−n>​N​​]),0≤n≤N−1

圆周共轭反对称

xca[n]=12(x[n]−x∗[<−n>N]),0≤n≤N−1x_{ca}[n]=\frac{1}{2}(x[n]-x^{*}[<-n>_N]), \, 0\leq n \leq N-1 x​ca​​[n]=​2​​1​​(x[n]−x​∗​​[<−n>​N​​]),0≤n≤N−1

例:考虑长度为444的有限长序列,0≤n≤30\leq n \leq 30≤n≤3:

u[n]={1+j4,−2+j3,4−j2,−5−j6}u[n]=\{1+j4,-2+j3,4-j2,-5-j6\} u[n]={1+j4,−2+j3,4−j2,−5−j6}

则

u∗[n]={1−j4,−2−j3,4+j2,−5+j6}u^{*}[n]=\{1-j4,-2-j3,4+j2,-5+j6\} u​∗​​[n]={1−j4,−2−j3,4+j2,−5+j6}

u∗[<−n>4]={1−j4,−5+j6,4+j2,−2−j3}u^{*}[<-n>_4]=\{1-j4,-5+j6,4+j2,-2-j3\} u​∗​​[<−n>​4​​]={1−j4,−5+j6,4+j2,−2−j3}

所以

ucs[n]={1,−3.5+j4.5,4,−3.5−j4.5}u_{cs}[n]=\{1,-3.5+j4.5,4,-3.5-j4.5\} u​cs​​[n]={1,−3.5+j4.5,4,−3.5−j4.5}

uca[n]={j4,1.5−j1.5,−j2,−1.5−j1.5}u_{ca}[n]=\{j4,1.5-j1.5,-j2,-1.5-j1.5\} u​ca​​[n]={j4,1.5−j1.5,−j2,−1.5−j1.5}

# 基于几何对称的分类

对称序列:

x[n]=x[N−1−n]x[n]=x[N-1-n] x[n]=x[N−1−n]

反对称序列

x[n]=−x[N−1−n]x[n]=-x[N-1-n] x[n]=−x[N−1−n]

由于NNN可以为偶数，也可以为奇数，所以存在四种类型的几何对称的定义。

考虑长度为555的序列

则其傅里叶变换为
