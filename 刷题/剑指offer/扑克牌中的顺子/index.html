扑克牌中的顺子

熊滔 2021-6-27 剑指offer About 1 min

从扑克牌中随机抽 555 张牌，判断是不是一个顺子，即这 555 张牌不是连续的。2−102-102−10 为数字本身，AAA 为 111，JJJ 为 111111，QQQ 为 121212，KKK 为 131313，而大小王可以看做是任意的数字。

我们可以把 555 张牌看做是 555 个数字组成的数组，大小王是特殊的牌，我们不妨定义为 000。

而判断这 555 张牌能不能够组成数字，因为大小王可以当做是任意的牌，所以我们就是判断间隔数和王牌的数目。什么叫间隔数，比如 2,32,32,3 之间没有间隔，间隔为 3−2−1=03 - 2 -1 = 03−2−1=0，2,52,52,5 之间的间隔为 5−2−1=25 - 2 - 1 = 25−2−1=2，之间空了两张牌。如果间隔数小于等于王牌的数目，那么因为王牌可以充当任意牌，可以组成一个顺子，但是如果间隔数大于王牌数，那么无论王牌充当什么都不能组成一个顺子。

另外，如果有对子，那么无论如何都不可能是顺子。

public static boolean isContinuous(int[] data) {
    if (data == null || data.length != 5) {
        return false;
    }
    // 首先对数组进行排序，前面出现的0表示王牌，后面的数表示其他牌
    Arrays.sort(data);
    // 王牌的数目
    int numberOfZero = 0;
    // 间隔数
    int numberOfGap = 0;
    // 统计王牌的数目
    for (int i = 0; i < data.length && data[i] == 0; i++) {
        numberOfZero++;
    }
    int small = numberOfZero;
    int big = small + 1;
    while (big < data.length) {
        // 如果有对子，那么不可能是数字你
        if (data[small] == data[big]) {
            return false;
        }
        // 统计间隔数
        numberOfGap += (data[big] - data[small] - 1);
        small++;
        big++;
    }
    // 如果王牌数目大于等于间隔数，返回 true
    return numberOfZero >= numberOfGap;
}
